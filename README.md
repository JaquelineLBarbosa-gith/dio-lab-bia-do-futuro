# 🎓 Diogo: O Seu Educador Financeiro com IA

Este projeto foi desenvolvido como entrega final do **Bradesco-GenAI-Dados-Bootcamp** na plataforma DIO.

## 🎯 Sobre o Projeto

Apresento o **Diogo**, um Agente de Inteligência Artificial criado para ser um educador financeiro paciente, imparcial e focado na literacia financeira para iniciantes. O seu objetivo é desmistificar conceitos complexos do mundo dos investimentos de forma didática e acessível, eliminando o "economês" complicado.

## 🛡️ Diferenciais e Regras de Segurança (Guardrails)

Diferente de assistentes genéricos, o Diogo foi desenhado com regras éticas e de segurança bastante rigorosas:
- **Ensina, mas não recomenda:** Ele explica como o mercado funciona, os prós e os contras, mas é **expressamente proibido** de dar dicas de compra ou venda de ativos.
- **Capacitação:** O objetivo é empoderar o utilizador para que ele próprio tome as suas decisões financeiras com segurança.
- **Segurança (Anti-Alucinação):** Se questionado sobre o futuro do mercado ou cotações, o Diogo admite a sua limitação e foca no ensino, evitando inventar respostas.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python
- **Interface Web:** Streamlit
- **LLM:** Ollama (modelo Llama 3)

## 📁 Estrutura do Repositório

- **`/docs`**: Contém toda a documentação do planeamento, incluindo o caso de uso, base de conhecimento, engenharia de prompts, métricas de qualidade e o guião da apresentação.
- **`/src`**: Contém o código-fonte da aplicação (`app.py`), responsável pela interface de chat simulada.
